# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2026-07-18

### Added

- Configurable HTTP methods, headers, request bodies, timeouts, and redirects
- Transport, HTTP status-family, response-header, and consistency diagnostics
- Human-readable CLI reports and structured JSON output
- Sensitive-data redaction and HTTP/HTTPS protocol restrictions
- PHPUnit test suite, PHPStan level 8, and GitHub Actions quality automation
- Client presentation in editable PowerPoint and PDF formats

### Quality

- 36 automated tests with 91 assertions
- CI coverage across PHP 8.2, 8.3, 8.4, and 8.5
